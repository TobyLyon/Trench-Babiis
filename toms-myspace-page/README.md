# TrenchBabii

TrenchBabii is a retro-web satire profile + tool suite (blinkies, Y2K UI, generators, and a token promo shell).

## Run locally

- Open a terminal in this folder and run a simple local server.
- Then visit `http://localhost:8000/index.html`.

Example:

```bash
python -m http.server 8000
```
